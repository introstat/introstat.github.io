Topic 06: Normal Distribution

Word Description
a. Shifting [b] The process of dividing all the values of a variable by a constant.
b. Rescaling [c] The purpose of this is to be able to compare two quantitative variables on different scales.
c. Standardizing [a] The process of adding or subtracting a constant from every value in a variable.
