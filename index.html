Electronic Assessment Materials for Introductory Statistics Courses

In this NSF-funded research project, we developed an electronic assessment model with vocabulary, clicker, and homework assessment questions for a broad range of undergraduate introductory statistics courses. Along with the assessment questions, we developed report program software incorporating the current state of data visualization research to automatically generate reports for students, course instructors, and supervisors, enabling rapid feedback on learning and instruction to all parties. This project extends previous work on course level student learning outcomes (GAISE College Report) to the topic level and develops assessments to measure these outcomes. These assessments can be used to find the relative difficulties students have in learning course outcomes, to obtain the most common patterns exhibited by students in their understanding of course learning outcomes, and to identify the most common problems students have in learning particular outcomes.

Topics

Our electronic assessment model divides the material in the introductory statistics course into 27 different topic areas covering content in descriptive statistics, data collection, probability and statistical inference. Coverage of each topic is self-contained, so instructors can choose only the topics covered in their specific course in any order. Topics are not structured around any textbook, but include a broad range of topics covered in many different general introductory statistics courses.

Report Software Package

We have written an R package that takes grade sheets from blackboard, and compiles them into reports. Short reports for each topic and section, and overall sections. Longer reports can be made across topics and sections. Reports tracking student performance can be made also. Reports are produced using the knitr package in R, and automatically clean the data, and flag particular aspects that might need some attention of the instructor or coordinator.

Obtaining the Materials

To request a copy of the materials described here, please email Amy Froelich. When your request is received and it is verified that you will use the materials for educational purposes, you will be given complete access to everything. If you wish to contribute you may request access to the private GitHub repository. For instructions on how to use GitHub, click here.

Getting started with GitHub

Step 1: Visit github.com then click "Sign up".

Step 2: Create a username and password, then select "Create an account" and choose the free plan.

Step 3: Once your is set up, email us at amyf@iastate.edu and request access to the introstat materials repository.

Step 4: If you are comfortable using the command-line, jump to the git command-line instructions below. If not, have no fear. GitHub has an easy-to-use GUI for both Windows and Mac. If you are a Windows user, go to windows.github.com. For Mac users, visit mac.github.com. Then download and install the application.

Step 5: Open the GitHub application and make sure to set up with your GitHub username and password before doing anything else.

Step 6: In the upper left corner of the UI, click the "+" icon and then click on the clone tab. Once you have been given access to the materials repository, you should see a repository in the list called materials under the introstat organization. We will notify you when this will become available. You should also receive an email from GitHub at that time.

Step 7: Click on the materials repository and choose a location on your computer where you would like to keep the files, then click "Clone".

Step 8: Enjoy! For more reading on how to use GitHub see GitHub bootcamp.

Command-line instructions

Step 1: Visit git-scm.com/downloads and download the appropriate version for your operating system.

Step 2: Set up Git using the instructions on this page.

Step 3: Once you have received a notification from us that you have been given access to the materials, cd to the directory where you would to keep the files.

Step 4: Enter git clone git@github.com:introstat/materials.git.

Step 5: Enjoy! For more information about how to use Git from the command-line, a good starting point would be git-scm.com/docs/gittutorial.
